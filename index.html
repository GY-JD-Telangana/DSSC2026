⚙️ Admin Setup Guide

Important Note:

Web browsers cannot directly read your local "Master List.xlsx" or securely write directly to a Google Sheet link without a middle layer. Follow these two steps to make this form completely functional.

Step 1: Updating the Participant List

  1. Open your Master List.xlsx and convert it to a JSON format (you can use free online Excel to JSON converters).
  2. Ensure the columns map to: name, role, dealership, department, xId.
  3. Open this HTML file in a text editor.
  4. Search for const participantsData = [ in the code below.
  5. Replace the sample data with your real JSON data.

Step 2: Connecting to your Google Sheet

To save data directly into your specific Google Sheet (1Bx-ZYsGSTstg4...), you must create an Apps Script endpoint.

  1. Open your Google Sheet.
  2. Go to Extensions > Apps Script.
  3. Delete any code there and paste the code below:
function doPost(e) {
  var sheet = SpreadsheetApp.openById('1Bx-ZYsGSTstg4ZiqIgdvby76Snu3YaPbsfAzHL_h7cQ').getActiveSheet();
  
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(["Timestamp", "Name", "Role", "Dealership", "Department", "X ID", "T-Shirt Size", "WhatsApp Number"]);
  }
  
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([new Date(), data.name, data.role, data.dealership, data.department, data.xid, data.tshirtSize, data.whatsappNumber]);
  
  return ContentService.createTextOutput(JSON.stringify({"status": "success"}))
    .setMimeType(ContentService.MimeType.JSON);
}
